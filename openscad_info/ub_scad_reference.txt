# UB.scad OpenSCAD Library Reference Guide

## Library Overview
**Author:** UBaer21  
**License:** CC0-1.0  
**Description:** A full 3D printing workflow solution for OpenSCAD v.21 and above  
**Usage:** `include <UB/ub.scad>`

UB.scad is a comprehensive 3D printing workflow library that brings tools for modifying and generating 3D objects, view helpers, and a selection of mechanical parts. The library is designed to be used without ever looking at the library's code itself, with extensive help systems and console feedback.

## When to Use UB.scad

### Perfect For:
- **Complete 3D printing workflows** - From design to print-ready models
- **Rapid prototyping** - Quick creation of functional parts with built-in tolerances
- **Mechanical parts** - Gears, bearings, screws, and assemblies
- **Educational projects** - Learning 3D modeling with guided examples
- **Production parts** - Professional-grade mechanical components

### Key Advantages:
- **Self-contained workflow** - Design, validate, and prepare for printing in one library
- **Built-in help system** - Extensive documentation accessible within OpenSCAD
- **3D printing optimized** - Automatic consideration of nozzle sizes, clearances, and tolerances
- **Console feedback** - Real-time information about what the library is doing
- **No dependencies** - Single file solution

## Installation and Setup

### Basic Installation
```openscad
// Single file approach - just include

// All examples and help available with this line
```

```openscad

// Workflow Settings
```


## Essential UB.scad Modules

### Mechanical Components

#### Gears (Complete Gear System)
```openscad
include <ub.scad>
/*[Gears]*/
z = 10;           // Number of teeth
f = 3;            // Teeth width divisor
modul = 2;        // Teeth size
h = 5;            // Height
w = 30;           // Helical teeth skew
achse = 3;        // Arbor diameter

// Create gear with UB.scad
CyclGetriebe(h=h, z=z, f=f, modul=modul, linear=false, w=w, achse=achse, help=true);

// Gear ratio calculation
function pitchcircle(z=z) = z*modul/f;

// Multiple gear arrangements
T(pitchcircle(z)*2) { 
    CyclGetriebe(h=h, z=z, f=f, modul=modul, linear=false, center=false, w=w, achse=achse);
    mirror([1,0]) CyclGetriebe(h=h, z=z, f=f, modul=modul, linear=false, center=false, rotZahn=-1, w=w, achse=achse);
}

// Internal/ring gears
T(pitchcircle(z)*4) rotate(180) { 
    CyclGetriebe(h=h, z=z*3, f=f, modul=modul, linear=false, center=false, d=pitchcircle(z*3)+modul, w=w, achse=achse);
    CyclGetriebe(h=h, z=z, f=f, modul=modul, linear=false, center=false, w=w, achse=achse);
}

// Rack and pinion
T(0, -pitchcircle(z)*2) {
    CyclGetriebe(h=h, z=z, f=f, modul=modul, linear=modul*2, center=false, w=w, achse=achse);
    CyclGetriebe(h=h, z=z, f=f, modul=modul, center=2, rotZahn=0, w=w, achse=achse);
}
```

#### Advanced Gear Features
```openscad
// Gear with locking mechanism and lightening
T(pitchcircle(z)*2, pitchcircle(z)+pitchcircle(z*3)/2) 
    CyclGetriebe(h=h, z=z*3, f=f, lock=5, light=5, modul=modul, w=w, achse=achse);
```

## Advanced UB.scad Workflow

### Complete Project Template
```openscad
include <UB/ub.scad>

/*[Hidden]*/
useVersion = 22.046;
designVersion = 1.0;

/*[Basics]*/
vp = false;          // View position mode
bed = false;         // Show print bed
pPos = [0,0];       // Print position on bed
info = true;        // Console information
nozzle = 0.4;       // Printer nozzle diameter

/*[Project Specific]*/
part_size = 20;     // Main part dimension
tolerance = 0.15;   // Mechanical tolerance

/*[Advanced]*/
$helpM = true;      // Module help
render_quality = 50; // Render quality

// Main design
T(printPos) {
    my_mechanism();
    
    if (info) {
        echo("Part size:", part_size);
        echo("Total height:", get_part_height());
        echo("Print time estimate:", get_print_time());
    }
}

// Print bed visualization
if (bed) {
    %cube([200, 200, 1], center=true);
}

module my_mechanism() {
    // Your design here using UB.scad modules
    CyclGetriebe(h=5, z=20, f=3, modul=1, achse=3);
}
```

### Multi-Part Assembly Example
```openscad
include <ub.scad>

// Assembly with automatic spacing for printing
parts_to_print = ["gear1", "gear2", "housing"];

for (i = [0:len(parts_to_print)-1]) {
    T([i * 50, 0, 0]) {  // Space parts for printing
        if (parts_to_print[i] == "gear1") {
            CyclGetriebe(h=5, z=15, f=3, modul=1.5, achse=4);
        } else if (parts_to_print[i] == "gear2") {
            CyclGetriebe(h=5, z=30, f=3, modul=1.5, achse=6);
        } else if (parts_to_print[i] == "housing") {
            housing_part();
        }
        
        // Part labels
        if (info) {
            echo(str("Part ", i+1, ": ", parts_to_print[i]));
        }
    }
}
```

## Troubleshooting Guide

### Common Issues and Solutions:

**"Too many console messages"**
- Set `helpsw = false` to reduce output
- Set `info = false` to disable general information
- Use specific help levels (1-5) instead of `true`

**"Parts don't fit together"**
- Check `nozzle` setting matches your printer
- Adjust `clearance` for your printer's precision
- Use UB.scad's automatic tolerance calculations

**"Gears don't mesh properly"**
- Ensure both gears use same `modul` value
- Check `f` (teeth width divisor) is consistent
- Use UB.scad's gear spacing calculations

**"Slow rendering"**
- Disable `help` modes during final render
- Use lower fragment counts for preview
- Let UB.scad auto-adjust fragment count

**"Can't find specific feature"**
- Use `helpsw = true` to see all available modules
- Check console output for module suggestions
- Try module_name(help=true) for specific help
